01Our Message

入門カリキュラムHTML・CSSを終えた方へ

受講者の皆さん、ここまで一旦お疲れ様でした。

スムーズに進めた方・時間がかかった方、何度も指摘をされて心が折れそうになった方、いろいろいると思いますが、まずはここまでこれたことに自信をもってください。（この課題でOKもらったら持ってください...!）
今までで、ポインターから指摘されたこと一つ一つが、必ず将来に生きることだと思っています。

まだまだ先は長いですが、本課題を終えることで、HTML・CSSの基本だけでなく、「調べ・考え・論じる」ことの重要性と、問題解決能力の根本を理解できたと思っています。

ここまで学んだことで、静的なサイト制作は一人称でできるようになったと思っていますが、まだまだ満足せずにさらなるスキルの向上を目指していきましょう！
